# Mesto

Mesto — учебный проект, выполненный в рамках курса по веб-разработке. Это интерактивная страница, где пользователь может редактировать профиль, добавлять карточки с фотографиями, ставить лайки и открывать изображения в увеличенном виде.

## О проекте

Проект представляет собой одностраничное приложение с адаптивной вёрсткой и подключением к серверу. Данные пользователя и карточки загружаются через API. Изменения профиля, добавление новых карточек, лайки и удаление карточек также сохраняются на сервере.

## Функциональность

- загрузка информации о пользователе с сервера;
- загрузка карточек с сервера;
- редактирование имени и описания профиля;
- обновление аватара пользователя;
- добавление новой карточки;
- удаление собственных карточек;
- постановка и снятие лайка;
- отображение количества лайков;
- открытие изображения в полноэкранном попапе;
- закрытие попапов по кнопке, оверлею и клавише Esc;
- валидация форм;
- отображение состояния загрузки при отправке данных.

## Работа с сервером

В проекте используется API Mesto. Через запросы к серверу выполняются основные действия: получение данных пользователя, получение карточек, изменение профиля, обновление аватара, добавление карточек, удаление карточек и работа с лайками.

## Установка и запуск

Склонировать репозиторий:

```bash
git clone https://github.com/TgBotStryzkov/mesto.git
```

Перейти в папку проекта:

```bash
cd mesto
```

Установить зависимости:

```bash
npm install
```

Запустить проект в режиме разработки:

```bash
npm run dev
```

Собрать проект:

```bash
npm run build
```

Опубликовать проект на GitHub Pages:

```bash
npm run deploy
```
